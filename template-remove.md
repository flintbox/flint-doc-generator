# Remove

***

Remove templates under template home.

## Command

```shell
flint template remove <template-name>... [--verbose | -v]
```

or

```shell
flint t r <template-name>... [--verbose | -v]
```

## Argument

`<template-name>...`

Templates to remove from template home.

## Option

`[--verbose | -v]`

Output more information.
